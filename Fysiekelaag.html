De Fysieke Laag

Processor

Een processor, ook bekend als een CPU, is een hardwareonderdeel dat verantwoordelijk is voor het uitvoeren en controleren van instructies. De werking van de CPU verloopt als volgt: eerst vindt de ophaalfase plaats, waarin de processor instructies uit het RAM-geheugen haalt met behulp van een programmateller. Daarna worden de instructies in de decodeerfase geanalyseerd. In de uitvoerfase worden de instructies vervolgens uitgevoerd. Ten slotte wordt de programmateller aangepast, zodat het proces opnieuw kan beginnen.

Werkgeheugen (RAM/ROM)

Het RAM-geheugen van een computer is een type geheugen dat de processor snel kan benaderen. In dit geheugen kan data in willekeurige volgorde worden gelezen of opgeslagen. Het ROM-geheugen, wat staat voor read-only memory, is een vorm van RAM-geheugen waarin de gegevens niet kunnen worden aangepast.

Permanent geheugen

Permanent geheugen is het type geheugen in een computer dat zijn gegevens behoudt, zelfs wanneer de computer is uitgeschakeld. Dit type geheugen wordt doorgaans gebruikt voor het opslaan van informatie die voor langere tijd moet worden bewaard. Voorbeelden van permanent geheugen zijn de harde schijf en solid-state drives (SSD's). De gegevens blijven opgeslagen omdat ze stroom ontvangen van het moederbord.

Randapparatuur

Randapparatuur verwijst naar hardware die specifieke taken uitvoert en de functionaliteit van een computer ondersteunt. Hoewel deze apparaten niet strikt noodzakelijk zijn voor de werking van de computer, maken ze het gebruik ervan aanzienlijk eenvoudiger en aangenamer. Voorbeelden van randapparatuur zijn een toetsenbord, een muis en een monitor.
