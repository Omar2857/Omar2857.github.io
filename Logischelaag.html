    • Operating system

    Een operating system, ofwel besturingssysteem, is een software die de hardware aanstuurt en bepaalt wat deze moet doen. Dit zorgt ervoor dat applicatieontwikkelaars dit niet zelf hoeven te programmeren. Voorbeelden van besturingssystemen zijn Windows, iOS, Android en Linux.

    talen/compilers

    Om te programmeren heb je een taal nodig waarin je de instructies kunt schrijven. Deze programmeertalen zijn "Turing-volledig", wat betekent dat ze in staat zijn om alle taken uit te voeren die een moderne computer kan uitvoeren. Voorbeelden van programmeertalen zijn Java, Python en HTML. Een compiler fungeert als een vertaler die de code van een programmeertaal omzet naar een uitvoerbaar programma.
